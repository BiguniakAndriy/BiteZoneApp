**BiteZone** is a fishing app that helps you **track your fishing sessions and manage your favorite waters and locations**.  

### Main Features
- Track fishing sessions: number of fish caught, total weight, trophy weight, duration, and notes.
- Save favorite water areas and specific locations within them.
- Add descriptions to sessions, waters, and locations.
- Save locations using GPS coordinates, so you can easily navigate back to them later.
- Fishing tools: interval timers for recasts, fish counters, and other useful utilities.
